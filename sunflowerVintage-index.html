Project:
Sunflower Vintage

Store build with Wordpress WooCommerce

Sunflower Vintage is not a commercial project. The website was created to demonstrate my skills in using Wordpress WooCommerce. The store sells used and branded Sunflower Vintage clothing.

Website contains a functioning e-mail contact section. The store is responsive on mobile devices. I used Astra theme with the use of several plugins such as: Spectra, Payment Plugins for PayPal WooCommerce, Converter for Media, Cookie Notice & Compliance for GDPR / CCPA, LiteSpeed ​​Cache, Inpost Delivery, Forminator ... and several others. The store has many options such as product filtering, payment gateways, and various delivery options. The user can create his own account. The store is safe, has good performance and SEO rating.
