Project:
Hoshizora

Restaurant build with React!

The Hoshizora restaurant website is designed as a modern application built with React JS and organized using the BEM methodology for CSS class management.

The project focuses on a component-based approach, allowing for easy and fast content management. Key information, such as the menu and dish descriptions, is stored within a structured object in the constants folder, making it simple to edit and update without modifying the source code of components.

To ensure smooth and appealing animations, the website uses the GSAP library with Observer and ScrollTrigger modules. These tools enable advanced control over visual effects during scrolling, enhancing interactivity and the visual appeal of the site.

Thanks to modern technology and a solid component structure, the Hoshizora website is flexible, ready for further development, and can easily integrate additional functionalities in the future.
